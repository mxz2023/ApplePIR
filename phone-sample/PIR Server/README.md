# Setting up a PIR server for URL filtering

Configure and run a PIR server to support a URL filter using Apple's open source container tool.

## Overview

As described in [Filtering traffic by URL](https://developer.apple.com/documentation/networkextension/filtering-traffic-by-url), a Network Extension URL filter uses a Private Information Retrieval (PIR) server as part of a complete URL filtering system.
The sample code project `SimpleURLFilter` contains a PIR server for use with the `SimpleURLFilter` app and its `SimpleURLFilterExtension` app extension.

See the article [Setting up a PIR server for URL filtering](https://developer.apple.com/documentation/networkextension/setting-up-a-pir-server-for-url-filtering) for more information on building, configuring, and running the PIR server.
